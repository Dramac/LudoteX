Aide — Tournois

Mode d'emploi pour organiser et suivre un tournoi.

Le cycle d'un tournoi

Un tournoi passe par quatre états : brouillon (en préparation, invisible du public) → inscriptions (ouvertes, visible et inscriptible) → lancé (mode de scoring choisi, parties en cours) → terminé. Les boutons d'état sont sur la page Gérer du tournoi.

Créer un tournoi

Depuis Tournois, touchez Nouveau tournoi. Renseignez au minimum un nom ; le jeu, la date, la durée, le nombre de places et l'emplacement sont facultatifs. Le tournoi est créé en brouillon : prenez le temps de tout régler, puis Ouvrir les inscriptions pour le rendre public.

Inscriptions et désinscription

Quand les inscriptions sont ouvertes, le public s'inscrit avec un simple pseudo. Un code de désinscription lui est alors affiché : c'est le seul moyen de se désinscrire. Aucune adresse e-mail n'est demandée ni conservée (l'envoi du code par e-mail viendra plus tard). Vous pouvez aussi ajouter ou retirer des participants à la main depuis la page Gérer (utile pour les inscriptions sur place).

Tournois par équipes

À la création, cochez Tournoi par équipes et indiquez le nombre de membres par équipe. L'inscription demande alors un nom d'équipe et la liste des membres ; c'est le nom d'équipe qui apparaît partout (matchs, classement), et un seul code de désinscription vaut pour toute l'équipe. Compatible avec tous les modes de scoring (une équipe compte comme un participant).

Pour la ronde suisse et l'élimination, vous pouvez aussi cocher Best of 3 au lancement : la saisie d'une rencontre se fait alors en manches gagnées (ex. 2–1) et le vainqueur en est déduit. Sans BO3, on désigne directement le vainqueur (ou un nul en suisse).

High score (points cumulés)

Le plus simple. Après le lancement, saisissez les points de chaque participant dans Saisir les scores (modifiable à volonté). Le classement se fait par points décroissants ; les ex æquo partagent le même rang.

Ronde suisse

Vous fixez un nombre de rondes au lancement. À chaque ronde, l'application apparie les joueurs de niveau proche (par points) sans jamais refaire jouer deux mêmes adversaires. Si le nombre de joueurs est impair, un joueur est exempt (bye) et marque comme une victoire — chacun son tour. Pour chaque rencontre, indiquez A gagne / match nul / B gagne ; quand tous les résultats d'une ronde sont saisis, touchez Générer la ronde suivante. Barème : victoire 1 point, nul 0,5, bye 1.

Round robin (championnat)

Idéal pour 4 à 8 joueurs : chacun affronte tous les autres une fois. Toutes les rondes sont générées dès le lancement ; si le nombre de joueurs est impair, chacun a exactement un tour de repos. Saisie des résultats comme en ronde suisse (même barème). La page publique montre le classement et un tableau des confrontations.

Élimination directe

Arbre classique : le perdant est éliminé. Si le nombre de joueurs n'est pas une puissance de 2 (4, 8, 16…), les mieux placés reçoivent un bye au premier tour. Désignez le vainqueur de chaque rencontre, puis Générer le tour suivant (demi-finales, finale…). Le vainqueur du tournoi s'affiche après la finale.

Suivi public

Dès le lancement, la page publique du tournoi affiche le classement (ou l'arbre) en temps réel. Inutile de rien publier : il suffit de partager le lien du tournoi.

Supprimer un tournoi

La suppression (page Gérer → Zone de danger) demande une double confirmation et efface aussi les inscriptions et les résultats. Elle est définitive.
