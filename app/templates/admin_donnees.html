❓ Avant d'importer ou de restaurer

Un import ne supprime aucun jeu, mais il écrase les fiches dont le code figure dans le fichier — colonnes vides comprises. Réimportez un fichier complet plutôt qu'un fichier partiel.

Une restauration remplace la totalité des données par le contenu de l'archive. Le réflexe qui évite tout : téléchargez d'abord une sauvegarde de l'état actuel, elle sera votre retour en arrière.
